Compensatory Eye Scans While Driving

My research using driving simulation has focused evaluating how the relationship between vision impairment (e.g., simulated central vision loss, real central vision loss, and real peripheral visual field loss) and hazard detection performance. I have also been helped develop a pipeline that automates processing of driving simulator and eye and head tracking data. Further, I have used the driving simulator to investigate change blindness to hazards.

Visual Attention to Object Features

How much does task relevance influence our ability to retrieve information of objects we are looking at? Are task irrelevant features of an object ignored, remembered coarsely, or are they remembered with the same strength as a task relevant feature? One of the difficulties in evaluating task irrelevant memories is that simply asking about that information makes it task relevant. Here, I've utilized techniques, such as the surpirse trial methodology, to ask these investigate these questions.

Visual Working Memory

My research focused on the testing and creation of a computational simulation of how visual information is bound to form a memory representation in visual working memory. This model, called the Binding Pool model, simulated encoding as the conjunction of the visual features that compose an object (e.g., color, size) and a token, the ability to individuate an object in memory from other objects in memory. The Binding Pool model provided a mechanism for simulating the trade-off between the number of objects maintained in memory and the quality of each memory representation
